Proyecto 3: Parkour Gems.

Descripción: Este es mi proyecto de Unity en conjunto con mi compañero Seba, el juego conciste en un parkour simple en el cual debemos juntar todas las gemas para ganar y si caemos al agua perdemos vida.

Descargar Unity

Informe: Para este proyecto utilizamos variables publicas para crear el movimiento de las cajas en el mapa. Tambien utilizamos un array que registra mediante un bucle las gemas vamos obteniendo. En el caso de las vidas son una variable que disminuye al tocar el collider del agua y aumentan al recolectar un corazon, esta variable al ser menor o igual a 0 activa el script de reinicio del juego. Informe Completo
